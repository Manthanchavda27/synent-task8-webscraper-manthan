# Web Scraper (Python)

A Python-based web scraper that extracts publicly available data from a website and displays structured information.

## Features
- Fetches website data using requests
- Parses HTML using BeautifulSoup
- Extracts quotes and author names
- Displays structured scraped data
- Beginner-friendly web scraping project

## Technologies Used
- Python
- requests
- BeautifulSoup (bs4)

## Target Website
https://quotes.toscrape.com

## How to Run
```bash
py web_scraper.py
